# graphql-react-todos

## Requirements
* Node
* Npm
* Docker
* Docker Compose

## Startup
1. Create `.env` file and add environment variables based on `.env-sanple` file.
2. Up services with: `docker-compose up -d`.
3. Install and run application by running `npm i && npm start` in the `app` folder.
